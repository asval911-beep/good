# نظام الكعبي لإدارة الالتزامات وعمليات التمويل الأهلي (Kaabi Money)

مرحباً بك! هذا النظام مصمم كليا ليعمل **بدون إنترنت (Offline-First)** ويمكن تشغيله مباشرة ومستقلاً من أي **ذاكرة فلاش (USB Flash Drive)** على أيّ جهاز كمبيوتر دون الحاجة لشبكة إنترنت أو حتى تثبيت برامج إضافية على الأجهزة المستهدفة.

---

## 🚀 تشغيل النظام مباشرة من الفلاش ميموري (بدون إنترنت أو خادم)

تم إعداد النظام بخاصية المسارات النسبية الذكية (`base: "./"`). بعد بنائه، يمكنك فتح التطبيق وتصفح كافة حسابات وعقود المدينين وحساب الفوائد وجدولة الأقساط بإحدى الطريقتين:

### طريقة التشغيل الفوري والسهل (الموصى بها):
1. قم بفتح مجلد **`dist`** بعد بنائه (أو انسخ محتويات مجلد `dist` كاملاً إلى الذاكرة الفلاشية).
2. انقر نقراً مزدوجاً على الملف: **`index.html`**
3. سيفتح النظام بالكامل في متصفحك (مثل Google Chrome أو Edge) بأداء كامل وبدون الحاجة لإنترنت أو تشغيل خوادم!

---

## 🛠️ كيفية بناء وتجهيز النظام لأول مرة (للمطورين أو من جهازك الخاص)

إذا أردت تحميل وتعديل النظام أو إعادة بنائه من الصفر على جهازك، اتبع الخطوات البسيطة التالية:

### المتطلبات المسبقة:
* تأكد من تثبيت بيئة **[Node.js](https://nodejs.org/)** (الإصدار 18 أو أحدث) على جهازك الشخصي.

### الخطوة 1: تثبيت الحزم البرمجية والاعتماديات
افتح منفذ الأوامر (Terminal) في مجلد المشروع الرئيسي، ونفّذ الأمر التالي:
```bash
npm install
```

### الخطوة 2: تشغيل وضع التطوير المحلي (اختياري)
لتشغيل النظام على خادم محلي وتجربة التعديلات الحية:
```bash
npm run dev
```
وافتح الرابط التالي في المتصفح: `http://localhost:3000`

### الخطوة 3: بناء نسخة مستقلة للنظام (الفلاش ميموري)
لبناء المجلد المحسن والمستقل بذاته بالكامل والمصمم للنقل للفلاشة، نفّذ الأمر التالي:
```bash
npm run build
```

بعد انتهاء العملية، سيتم توليد مجلد جديد باسم **`dist`**. هذا المجلد هو كل ما تحتاجه للتشغيل المستقل.

---

## 📑 الخصائص والمميزات المدعومة للتشغيل الفردي والمستقل:

* **حفظ تلقائي فوري وتراكمي (Instant Auto-Save):** يقوم النظام بحفظ وتحديث كافة أسماء المدينين، تفاصيل المبالغ، التواريخ، التواقيع الحية، جدول الأقساط، والمرفقات تلقائياً في ذاكرة المتصفح المحلية لفتحها من نفس الجهاز عند الإغلاق وفتحه مجدداً.
* **الأرقام الإنجليزية المعتمدة:** يتم استخدام الأرقام القياسية الغربية (0-9) لتسهيل القراءة وتوحيد طباعة التقارير المحاسبية وسندات المقاصة وعمليات الحساب بالكامل.
* **إدارة وتصدير النسخ الاحتياطية:** تتوفر أزرار مدمجة واحترافية في شريط التوقيت العلوي للبرنامج للقيام بـ:
  * **تصدير نسخة احتياطية:** لتنزيل ملف احتياطي خارجي بصيغة `.json` يضم كل كبيرة وصغيرة بالبرنامج لحفظه كملف على الفلاشة أو إرساله لجهاز آخر.
  * **استيراد نسخة:** لاستعادة السجلات والتواقيع بالكامل على أي جهاز جديد بلمحة بصر.
  * **تصفير النظام:** لإعادة ضبط المصنع ومسح السجلات من المتصفح في حال تسليم النظام لشخص آخر.
  * **حفظ السند كصورة:** إمكانية تنزيل صورة السند المالي المختوم والموقع بجودة عالية للتخزين الورقي مع التوقيعات الحية للمدين المعتمد.
